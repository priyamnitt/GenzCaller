# GenzCaller Application

A modern calling application built with Next.js and Prisma.

## Features

- Real-time communication
- User authentication
- Database integration with Prisma
- Modern UI/UX design

## Prerequisites

Before you begin, ensure you have installed:
- Node.js (v14 or higher)
- npm or yarn
- PostgreSQL database
